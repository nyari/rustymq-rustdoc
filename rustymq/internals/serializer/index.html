rustymq::internals

Module rustymq::internals::serializer[−][src]

Serializer module

Summary

This module contains a basic serialization functionality used internally by RusyMQ But it can be used externally for serialization as a basic tool for serialization.

Details

The reason not to use a serialization library was to keep the dependencies of the library minimal and also to have a simple and fast binary serialization that also supports big and little endiannes
